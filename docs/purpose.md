# 概要

エンジニアのキャリア実現度を最大化できれば、それは日本の技術革新を間接的に支えることになり、日本の技術革新が促進されれば、社会の豊かさを底上げすることに資する。

## なぜこのパーパスなのか

### ポイント01：「自社ならでは」のメッセージになっているか

→**エンジニア(求職者)**から見て価値のあるサービスにしたい。  
　それが最終的に求人企業への価値をも生むこととなる。というのが当社の基本スタンス。  
　(**競合他社の大半はお金を払ってくれる”求人企業”の方ばかりを向いている**。求職者の中長期キャリアに真剣に向き合っているエージェントはほぼ皆無である。コンサルタント個別では求職者の方を向いている例は稀にみられる)

### ポイント02：企業の現状にフィットした未来視点で考えられているか

→当社はエンジニアのスキルセット、各スキルのレベル、その他の業務経験、中期的にどんなエンジニアなりたいのか、(**技術、個性、想い**)  
　を把握理解している。  
　だからこそ単なる転職支援ではなく、その人のエンジニア人生全体にとってポジティブな効果を及ぼす  
　”**エンジニアのキャリア実現度を最大化させるキャリア支援**”が可能となる。  
　(人体や病理についての最新の知見をupdateし続けている医者だからこそ精度の高い**診断**ができ、精度の高い診断ができるからこそ精度の高い**処方**ができる)　

### ポイント03：ワクワクを生む表現になっているか

→エンジニア一人ひとりのキャリア実現度を最大化するからこそ、その積み重ねが日本の産業全体の実力を高める。  
　だからこそ、我々の仕事は間接的に、ときには直接的に日本のテクノロジー発展に貢献し、日本全体の持続的な豊かさを底上げすることにつながっている。

## パーパスの構成要素とシステム実装への反映

エンジニアのキャリア実現度を最大化することで、  
（手段=How）

**【システム実装への反映：マッチングロジック】**  
本システムのスコアリングは「現状維持」ではなく「成長（Upskilling）」を最優先します。  
その人が今できること（Source）よりも、少し高いレベル（Target）への挑戦を「100%超のスコア」として推奨します。  
逆に、成長につながらないオーバースペックな求人は、たとえ即戦力であっても「減点（-50）」として排除します。  
これが「キャリア実現度の最大化」をシステム的に担保する仕組みです。

日本の技術革新を支え、  
（作用=effect）

社会の豊かさの底上げを図る。  
（目的=goal）

## 将来のビジョン：近未来のキャリア支援ビジネスの在り方

「時価総額世界一歴代4社に共通する復活の要諦」を人材紹介事業（グローバル・リクルートメント業界）に応用し、業界の変革シナリオを以下にまとめます。

### **🔄 「復活の要諦」× 人材紹介事業の変革モデル**

| 復活の原則 | 人材紹介業への応用 | 未来に向けた方向性 |
| :--- | :--- | :--- |
| **🎯 ビジョンの再構築** | 「求人を埋める」から「人的資本の戦略投資支援」へ | 人事戦略・組織開発のパートナーとして、採用の真の理由を追求する。 |
| **🔄 ポートフォリオ転換** | 手数料・単発モデルからの脱却 | サブスクリプション型契約、DX人材特化、教育・研修（リスキリング）との一体提供。 |
| **🧠 リーダーシップ刷新** | エージェント中心から「AI×コンサル」型HRテック企業へ | テクノロジーを起点に、ミッションに基づいた新たな採用手段を創出する。 |
| **🧩 柔軟な組織と外部連携** | SaaS・教育機関・副業プラットフォーム等とのハブ化 | 「雇用をつなぐインフラ企業」として外部エコシステムと連携する。 |
| **📈 政策・制度面との連動** | 雇用流動化や人的資本開示義務への適応 | 国や企業の「雇用調整支援機能」を担う公共性の高いポジションへ。 |

---

### **🔮 次世代の人材紹介業：具体的な変化像**

#### **1. 紹介から「キャリアデザイン」へ**
- 単なる人材配置に留まらず、キャリア・人生・スキルのアーキテクト（建築家）として関与。
- 例：「あなたのキャリアを“AI時代の武器”にする」伴走型支援。

#### **2. 「HRインテリジェンス」の提供**
- データとAIを活用し、採用成功率や定着予測（Predictive Hiring）を提示。
- 特に採用失敗コストを下げる指標を提供し、「ではどうすれば良いのか？」という方針・手段・戦術を案出する。

#### **3. 「スキル主権社会」への対応**
- 職種ではなく「スキル単位」で評価・提案するモデルを構築。
- スキルの“証明”と“マッチング”を補助する独自のプラットフォーム機能を備える。
- ※LinkedInやDegreed等の既存プラットフォームに依存せず、独自の価値を提供。
