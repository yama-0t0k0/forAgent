ドキュメントについて

本ドキュメントでは、各サービスごとに最適化されたカラーパレットを使用して画面遷移を定義しています。各ノードの色は、その画面が持つ「役割」や「状態」を直感的に示しています。

Screen (画面)

基本画面。サービスごとのメインカラーで着色。

Modal (モーダル)

背景色が薄く、二重線で表現されるオーバーレイ。

View State (内部)

点線枠の領域。同一ページ内での表示切り替え。

Action (遷移)

実線は順方向、点線は戻る動作を意味します。

補足事項・共通挙動

Shared Components

GenericRegistrationScreen

全てのアプリで使用される共通の登録・編集画面です。図中の白背景ノードはこの画面を指します。

Progress State

Under Construction

開発中の機能は視覚的に薄く表現されており、将来的な拡張性を示唆しています。

Interaction

Modal Overlay

二重線の枠（モーダル）は、操作が完了するまで親画面を「ロック」するオーバーレイを表します。
