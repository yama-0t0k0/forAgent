BE リファクタリング計画

Cloud Run (Pure Dart) への集約、セキュリティ強化、および Full-stack Dart サーバーへの進化プロセス。

戦略目標 (Strategic Goals)

API の統一

全てのクライアントリクエストを唯一の API サーバー (`apps/backend`) で受け、認可とロジックを集中管理します。

高速な起動とスケーリング

AOTコンパイルされた Dart ネイティブバイナリにより、ミリ秒単位のコールドスタートを実現します。

Next Step

  • ・OpenAPI / Swagger によるドキュメント自動生成

技術的負債の解消方針

  • 共通ライブラリのバージョン競合

    `melos` を使用し、モノレポ内のパッケージ間の pubspec 整合性を自動監視します。

  • 権限のハードコード排除

    コード内の `if (isAdmin)` ではなく、ミドルウェア層でのデコレータによる認可を徹底します。
