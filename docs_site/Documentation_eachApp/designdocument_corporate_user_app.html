企業ユーザー用アプリ
corporate_user_app

企業の技術ブランディング、求人掲載、および採用情報の管理を行うための法人向けプラットフォーム。

システム構成

UI 構造

  • - CompanyPageScreen: メインダッシュボード。カスタムボトムナビゲーションを搭載。
  • - TechStackView: 企業が保有・推奨する技術スタックの可視化。
  • - CorporateImageEditScreen: ヘッダー画像/プロフィール画像の編集。

データ永続化

共有コンポーネント `GenericRegistrationScreen` を介して `corporates` コレクションへの書き込みを実行。`Company` モデルによりデータ型を担保します。

企業用アプリは、Sharedコンポーネントをベースにしつつ、カスタムナビゲーションによりドメイン特化のUXを提供しています。

主要機能

技術ブランディング

企業が現在使用している技術スタックをバッジ形式で公開し、最適なエンジニアターゲットにアプローチ。

求人管理機能

`job_description` アプリと連携し、自社の求人情報を一覧管理。募集要項の編集を実行可能。

つながり管理

エンジニアユーザーとのコンタクト状況をトラッキング（将来拡張）。
