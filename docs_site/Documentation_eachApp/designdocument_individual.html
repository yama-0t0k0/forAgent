個人ユーザーアプリ
individual_user_app

エンジニア本人が自身のキャリア資産を可視化し、成長の軌跡を管理するためのメインアプリケーション。

プロジェクト概要

基本スペック

  • Framework: Expo (React Native)
  • Shared Module: common_frontend
  • Data Source: Firestore + JSON Fallback

設計目的

個人のプロフィール、スキル、および志向をヒートマップとして可視化。モデル・ファーストな設計により、公開/非公開データの透過的な結合を実現します。

モデル利用の徹底 (Model-First)

`User.fromPublicPrivate()` メソッドを介して、Firestore上の分散データを一つのエンティティとして扱います。生データへの直接アクセスは禁止です。

🆕 アプリケーション・モード (App Mode)

`LaunchController` を通じて、エントリポイントを切り替えて起動可能です。

1. Default モード (Full App)

通常のマイページ全機能を提供。`User` モデルを介して加工・統合されたデータを表示します。

src/apps/FullApp.js

2. Registration モード

登録特化型の独立ページ。`yama` ブランチの複雑な加工をバイパスし、純粋な JSON 入力を可能にします。

src/apps/RegistrationApp.js
