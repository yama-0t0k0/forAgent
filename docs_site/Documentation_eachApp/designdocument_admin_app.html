ヒートマップ要約ロジック

全エンジニアのスキルタグ・経験データを集計し、現在の技術トレンドをヒートマップ形式で可視化します。

Logic Implementation: `HeatmapCalculator.js`

  • 1. Firestore から `individual_user` を一括取得。
  • 2. `skillTags` を正規化 (Normalize) し、出現頻度をカウント。
  • 3. Recharts 経由で、技術ドメインごとの強度をグラデーション表示。

特権アクセス制御 (RBAC)

  • Only Admins Allowed

    Custom Claims `role: admin` が付与されたアカウントのみが、API 経由で全件データ取得を許可されます。

  • Audit Logging

    管理者が個人情報を閲覧・編集した際、`audit_logs` コレクションに操作ログが自動記録されます。
