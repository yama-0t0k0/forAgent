microCMS セットアップ

LaT PJ の LP コンテンツ管理基盤。API スキーマ定義と環境設定のガイド。

API スキーマ定義 (`lp_home`)

フィールドID 表示名 種類 必須
title タイトル テキスト ✅
body 本文 リッチエディタ ✅
thumbnail サムネイル 画像 -
is_premium_only Premium限定 真偽値 ✅
target_company_id 対象企業ID テキスト -

環境変数の設定

`apps/functions/.env` (git対象外) に以下の値を設定してください。

MICROCMS_SERVICE_DOMAIN=latlp
MICROCMS_API_KEY=your_api_key_here

テストデータの作成推奨

公開用 (Public)

`is_premium_only: OFF` で作成。ゲストユーザーの閲覧確認用。

限定用 (Premium)

`is_premium_only: ON` で作成。認証・認可ロジックの判定検証用。
